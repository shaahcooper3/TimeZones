THIS IS YOUR TAGS SECTION:

head: Creates heading
body: Creates body
title: Creates new title
h1: Creates new heading
marquee: Moves text across row on screen
!-- --: Comments out lines (VIEW EXAMPLE IN VS CODE)
br: Prints on a new line (DOES NOT HAVE A END TAG)
strong: Puts element in bold
em: Puts element in emphasis
a: Anchor tag is used to define the beginning and end of a hypertext link. div: Defines a division or a section in an HTML document
ul: Represents a list of items, where the order of the items is not important
ol: Designates an ordered list li: Represents a list item in ordered and unordered lists. The "li" tag is placed inside either an "ol" or "ul" to represent each individual item
